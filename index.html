🔍 Root Cause Framework

When Kooner is not getting the work, the cause falls into one of two buckets. The response is different for each.

⚡ Performance Gap

Response time, repair quality, completion rate, pre-auth process. DSP stopped calling because of a friction point. Fix: operational improvement.

🤝 Relationship Gap

DSP had a bad experience, drifted to another vendor, or never had a strong connection to begin with. Fix: direct outreach, accountability, trust-building.

Key: Diagnose which bucket each station falls into before deciding the response. Sending a relationship pitch to a performance problem — or vice versa — won't work.

📌 Guiding Principles
Diagnose before acting

Every leaking station needs a root cause diagnosis — performance gap or relationship gap — before the response is decided. The wrong fix wastes time and can make things worse.

Fix capability first

HVAC, brake/air system, and diagnostics gaps need to be closed operationally before DSP outreach on those repair types can succeed. A pitch without the capability to back it up erodes trust further.

Measure progress weekly

WK11 is the baseline: 58 ROs, 9 stations. Pull leakage data every Monday and track whether the number is moving. Stations where outreach is active should show improvement within 2–3 weeks.
